# Britannia Production Line Scheduling Optimization

This project optimizes the production scheduling of Britannia biscuits using
Linear Programming and Google OR-Tools.

## Biscuit Variants Used
- Marie Gold
- Good Day
- Treat Choco

## Objectives
- Minimize total production makespan
- Reduce changeover losses
- Increase line utilization

## Tools Used
Python, OR-Tools, Linear Programming, Pandas, Excel
